# 演示文档索引

本目录保存面向答辩、面试和现场展示的演示 Runbook（运行手册）。它和 `docs/interview/` 的区别是：这里更关注“现场怎么打开、怎么刷新、怎么讲”，`docs/interview/` 更关注“项目怎么背、怎么答深挖问题”。

- `audit-console-demo.md`：GS-TBK（Group Signatures with Time-bound Keys，带时间绑定密钥的群签名方案）身份监管审计台演示脚本，覆盖公网展示、VM（Virtual Machine，虚拟机）内网展示、DNS（Domain Name System，域名系统）注意事项、数据刷新命令和讲解顺序。
